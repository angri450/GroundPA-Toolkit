# 分组散点图

## 用户想要什么

三个处理组的株高和产量的散点图，用不同颜色区分处理组。

## 数据

```json
{
  "groups": [
    {"name": "CK", "values": [{"x": 80, "y": 5.2}, {"x": 82, "y": 5.1}, {"x": 79, "y": 5.3}]},
    {"name": "N1", "values": [{"x": 95, "y": 6.1}, {"x": 98, "y": 6.3}, {"x": 93, "y": 5.9}]},
    {"name": "N2", "values": [{"x": 110, "y": 7.5}, {"x": 108, "y": 7.3}, {"x": 112, "y": 7.6}]}
  ]
}
```

## 做了什么

```powershell
nong chart scatter groups.json -o scatter.png --json
```

## 结果

散点图输出：X 轴=株高，Y 轴=产量，三个颜色分别对应 CK、N1、N2。

## 关键点

- 散点图的 values 是 `{x, y}` 对象数组，不是单一数字数组。和 bar/line 的格式不同。
- 每个 group 用不同颜色自动区分。
- 如果要做回归线或拟合，当前 `chart scatter` 不支持——这是功能缺口，如实告诉用户。
