# 直方图数据分布检查

## 用户想要什么

有 50 个玉米株高测量值，想看数据分布是否正态、有没有双峰。

## 数据

```json
{"株高":[180,185,172,190,178,182,195,170,188,175,168,192,180,183,177,186,174,191,169,184,179,187,173,193,171,181,176,189,166,194]}
```

## 做了什么

```powershell
# 默认 20 个 bin
nong chart histogram data.json -o height_hist.png --title "玉米株高分布" --xlabel "株高 (cm)" --ylabel "频数" --json

# 或者指定 bin 数
nong chart histogram data.json -o height_hist_10.png --bin-count 10 --title "玉米株高分布" --xlabel "株高 (cm)" --json
```

## 结果

直方图输出：X 轴株高范围，Y 轴频数，大致呈单峰正态分布，没有明显双峰或偏态。

## 关键点

- 直方图会把所有组的数值合并在一起画。如果你有多个处理组想分开看分布，用箱线图。
- `--bin-count` 控制柱子数量。默认 20。太少会掩盖细节，太多会误导。
- 直方图不需要统计检验。如果你要检验正态性（Shapiro-Wilk 等），当前 chart 命令还不支持——如实告诉用户。
- 数据格式和 boxplot 一样：`{"组名": [值1, ...]}`。多组数据全部合并到一张直方图。
