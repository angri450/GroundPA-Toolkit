# 箱线图处理组分布对比

## 用户想要什么

有四个氮肥处理组（CK、N1、N2、N3）的产量数据，想看各组的分布范围、中位数和离群值。

## 数据

```json
{"CK":[5.2,5.1,5.3,5.0,5.2],"N1":[6.1,6.3,5.9,6.2,6.0],"N2":[7.5,7.3,7.6,7.4,7.2],"N3":[7.0,6.9,7.1,6.8,7.0]}
```

## 做了什么

```powershell
nong chart boxplot data.json -o boxplot.png --title "氮肥处理产量分布" --ylabel "产量 (kg)" --json
```

## 结果

输出 `boxplot.png`：每个处理组一个箱子，显示 Q1、中位数、Q3、上下边缘。N2 组分布最高，CK 组最低，组内变异较小。

## 关键点

- 数据格式与其他 chart 命令一致：`{"组名": [值1, ...]}`。
- 箱线图不需要做 ANOVA——它就是可视化分布对比。如果要做统计检验，先 `chart anova` 再 `chart duncan`。
- 箱线图只能看到分布趋势，不能替代显著性检测。图中异常值不代表显著异常。
- 如果组数很多（>10），箱线图会变挤，考虑用柱状图 + error bar 替代。
