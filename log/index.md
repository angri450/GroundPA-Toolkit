# log index

Nong.Toolkit.Net development records are organized by purpose:

- [plans](plans/index.md): planned or approved work.
- [changelog](changelog/index.md): completed implementation changes.
- [debug](debug/index.md): audits, failures, reviews, and user feedback.
- [guidance](guidance/index.md): reusable development guidance.

Generated HTML reports belong in `log/reports/`.
